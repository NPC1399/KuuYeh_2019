How to Flash AS72xx Firmware using this Firmware Update GUI 

1. An active TCL environment is necesarry for using this GUI. If you don't have the ActiveState TCL Run-Time Setup on your PC then please start the Run-Tme Setup before starting  ths GUI

2. The GUI only works in UART communication mode. Therefore, if necessary then switch the test board to UART and start the GUI

3. Update the firmware and don't forget a switched UART to reswitch after the firmware upgrade 

4. More details are given in the application note.
